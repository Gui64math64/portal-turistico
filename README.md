# Projeto turismo

Projeto de site desenvolvido com HTML e CSS, com o objetivo de apresentar destinos turísticos e culinaria brasileiros

---

## Descrição

---

### Funcionalidades

- Página inicial com apresentação e links para as outras páginas
- Página de destinos turísticos
- Página de gastronomia
- Página de contato com formulário, gmail e whatsapp
- Menu de navegação funcional com destaque na pagina ativa
- Layout responsivo com flex
- Cards interativos com efeitos hover

---

### Destinos apresentados

- Rio de Janeiro
- Fernando de Noronha
- Salvador
- Foz do Iguaçu
- Lençóis Maranhenses
- São Paulo

---

### Recursos visuais

- Imagens
- Efeitos de hover na navegação e nos cards
- Transições suaves na navegação e nos cards
- Layout organizado em seções e cards
